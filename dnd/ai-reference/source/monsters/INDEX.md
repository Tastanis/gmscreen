## Monster JSON

After finding the monster text, use `../../vtt-json/INDEX.md` and `../../../strixhaven/monster-creator/MONSTER_JSON_IMPORT_TEMPLATE.md` for the current import shape. Do not infer fields from the book text when the code already defines the schema.
